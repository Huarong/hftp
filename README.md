### executive file

Executive binary file are in directory `bin`:

- client
- server-iomultiplexing
- server-multiprocess
- server-multithread

### Root permission is required to execute server:

sudo bin/server-iomultiplexing
sudo bin/server-multiprocess
sudo bin/server-multithread


### Configurations

#### Account

user: anonymous
password: anonymous

#### Port
21

### Supported coomands

- open
- user
- pass
- pwd
- get
- put
- exit
